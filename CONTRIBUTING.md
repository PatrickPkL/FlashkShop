# Guia de Contribuição

Obrigado pelo interesse em contribuir com o FlaskShop!  
Como este é um projeto de portfólio/template comercial, contribuições externas são bem-vindas na forma de **sugestões e reports**, não de pull requests diretos.

---

## 🐛 Reportando Bugs

1. Verifique se o bug já foi reportado nas [Issues](../../issues)
2. Abra uma nova issue com o template abaixo:

```
**Descrição do problema:**
**Passos para reproduzir:**
**Comportamento esperado:**
**Ambiente:** (OS, versão do Python)
**Prints (se houver):**
```

---

## 💡 Sugerindo Melhorias

Abra uma issue com o prefixo `[SUGESTÃO]` no título e descreva:
- Qual problema a melhoria resolve
- Como você imaginaria a implementação
- Se possível, exemplos de referência

---

## 📐 Padrões do Projeto

Caso contribuições de código sejam abertas no futuro:

- **Python:** seguir PEP 8
- **HTML/CSS:** manter o sistema de variáveis CSS (`:root`) do `base.html`
- **Commits:** mensagens em português, no imperativo — ex: `Adiciona filtro por categoria`
- **Branches:** `feature/nome-da-feature` ou `fix/nome-do-bug`

---

## 🤝 Código de Conduta

Seja respeitoso. Issues agressivas ou spam serão fechadas sem resposta.
